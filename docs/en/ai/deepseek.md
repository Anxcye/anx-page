# Get DeepSeek Configuration

## 1. Get API Key

1.  Visit [DeepSeek Open Platform](https://platform.deepseek.com/api_keys).
2.  Log in to your DeepSeek account.
3.  Click **Create API Key**.
4.  Enter a name and click **Create**.
5.  Copy the generated API Key.

## 2. Confirm URL

DeepSeek's API is compatible with OpenAI format, the address is:
`https://api.deepseek.com`

## 3. Select Model

Main models include:
-   `deepseek-chat` (DeepSeek-V3)
-   `deepseek-coder` (DeepSeek-V2.5, to be merged)
-   `deepseek-reasoner` (DeepSeek-R1)

Please visit [DeepSeek Docs](https://platform.deepseek.com/api-docs/) to confirm the latest model names.

## 4. Fill in Configuration

Back to the AI settings page of Anx Reader:
-   **URL**: Enter `https://api.deepseek.com`
-   **API Key**: Enter the Key obtained in step 1
-   **Model**: Enter the model name selected in step 3
