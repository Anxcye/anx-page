# DeepL Translation Service

DeepL is widely considered the world's best machine translation service, providing high-quality translations.

## Free Quota

DeepL offers a specific **DeepL API Free** plan which allows for **500,000 characters per month** for free.

## Configuration Guide

1.  **Sign up for DeepL API**:
    *   Visit the [DeepL API page](https://www.deepl.com/pro-api).
    *   Click on "Sign up for free" under the **DeepL API Free** plan.
    *   Complete the registration process. *Note: A credit card is usually required for identity verification, even for the free plan, to prevent abuse.*
2.  **Get your Authentication Key**:
    *   Log in to your DeepL account.
    *   Go to your "Account" summary page.
    *   Find the "Account management" or "API Usage" tab.
    *   Copy your "Authentication Key for DeepL API".
    *   *Tip: Free API keys usually end in `:fx`.*
3.  **Configure in App**:
    *   Open Anx Reader Translation settings.
    *   Select "DeepL" as the Full Text Translation service.
    *   **API URL**: For the free plan, ensure the URL is `https://api-free.deepl.com/v2/translate`. (This is usually the default in the app).
    *   **API Key**: Paste your Authentication Key (ending in `:fx`).
