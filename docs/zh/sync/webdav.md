# WebDAV 同步

Anx Reader 支持使用 WebDAV 协议进行多端数据同步。以下是配置和使用的详细步骤。

## 同步流程

我们使用 WebDAV 进行数据同步。假设你有两个设备：**设备 A**（有数据的设备）和 **设备 B**（全新的设备）。

### 第一步：在设备 A 上配置并上传

1.  **打开设置**：进入 `设置` -> `更多设置` -> `同步` -> `WebDAV`。
2.  **填写配置**：点击进入后，你会看到三个配置项：`URL`、`用户名`、`密码`。请填入你的 WebDAV 服务信息。（如何获取这些信息请参考下方 [WebDAV 服务参数获取](#webdav-服务参数获取) 章节）。
3.  **测试连接**：点击“测试链接”按钮，确保配置正确且能连接成功。
4.  **启用并上传**：
    *   点击“启用 WebDAV”。
    *   在弹出的选项中选择 **“上传到 WebDAV”**。这会将当前设备（设备 A）的数据上传到云端。
5.  **查看进度**：你可以点击书架页右上角的同步按钮查看同步进度。

### 第二步：在设备 B 上配置并下载

1.  **配置 WebDAV**：使用设备 B，同样进入 WebDAV 设置页面，填入相同的 WebDAV 配置信息并测试连接成功。
2.  **启用并下载**：
    *   点击“启用 WebDAV”。
    *   在弹出的选项中选择 **“从 WebDAV 下载”**。这会将云端的数据下载到当前设备（设备 B）。

### 第三步：自动同步

完成上述步骤后，两个设备的数据已经保持一致。之后，应用会根据数据更新时间自动进行同步，保持多端数据实时一致。

## WebDAV 服务参数获取

以下是两个常用 WebDAV 服务的参数获取方法：

### 坚果云 (Jianguoyun)

1.  登录坚果云网页版。
2.  点击右上角账户名，选择 `账户信息` -> `安全选项`。
3.  在“第三方应用管理”中，点击 `添加应用`，生成一个应用密码。
4.  **服务器地址 (URL)**：`https://dav.jianguoyun.com/dav/`
5.  **用户名**：你的坚果云注册账号（邮箱）。
6.  **密码**：刚才生成的应用密码（不是你的登录密码）。

### InfiniCloud

1.  登录 InfiniCloud 网页版 (My Page)。
2.  在页面中找到 `Apps Connection` (WebDAV) 部分。
3.  勾选 `Turn on Apps Connection`。
4.  **服务器地址 (URL)**：页面上显示的 URL (例如 `https://TeraCLOUD.jp/dav/` 或 `https://s-us.c.teracloud.jp/dav/` 等，请以你页面显示的为准)。
5.  **用户名**：你的 User ID。
6.  **密码**：Apps Connection Password (显示的那个 WebDAV 专用密码)。
