# 简介

Anx Reader 是一款专为爱书人精心打造的电子书阅读器。它集成了强大的 AI 功能，并支持多种电子书格式，致力于让阅读变得更智能、更专注。凭借现代化的界面设计，我们致力于为您带来纯粹的阅读乐趣。

## 主要功能

*   **强大的 AI 能力**: 集成 AI 服务，提供内容摘要等功能，让阅读更智能。
*   **多格式支持**: 支持 EPUB, MOBI, TXT 等多种常见电子书格式。
*   **跨设备同步**: 无论是在 iOS, Android, Windows 还是 Mac 上，都能无缝同步您的阅读进度、笔记和书籍。
*   **阅读统计**: 提供详细的周、月、年阅读统计和直观的热力图，帮您追踪阅读习惯。
*   **高度定制**: 支持自定义主题、行距、边距和字体样式，打造专属阅读环境。

## 社区项目

以下项目由社区维护：
*   **Calibre 插件**: [anx-reader-calibre-plugin](https://github.com/ptbsare/anx-reader-calibre-plugin) - 让您可以直接从 Calibre 管理 Anx Reader 电子书库。
*   **Web 库管理**: [anx-calibre-manager](https://github.com/ptbsare/anx-calibre-manager) - 一个现代化的 Web 应用，用于管理电子书库，支持 WebDAV。
